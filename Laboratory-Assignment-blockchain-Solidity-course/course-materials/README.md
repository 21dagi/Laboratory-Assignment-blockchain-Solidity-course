# Solidity & Web3 Learning Assets

A curated collection of essential external documentation and tooling references for this smart contract development space.

## Reference Materials & Guides

* **Alchemy Solidity Slide Decks**: [presentations repository](https://github.com/alchemyplatform/learn-solidity-presentations) - Reference slides for the Solidity curriculum.
* **Marp Presentation Engine**: [marp.app](https://marp.app/) - Markdown-based presentation tool.
* **Foundry Development Guide**: [Foundry Book](https://book.getfoundry.sh/) - Complete handbook for testing and compilation with Foundry.

## EVM & Low-Level Reference

* **EVM Opcodes Interactive Guide**: [evm.codes](https://www.evm.codes/) - Comprehensive interactive explorer for Ethereum Virtual Machine instruction set.
